# Kage Bunshin Hand Sign Guide

How to position your hands so the clone effect triggers.

---

## Overview

You need **BOTH hands** in frame, forming a **cross (+) shape**:

- **Left hand**: Index + middle fingers UP, ring + pinky FOLDED
- **Right hand**: Same finger position
- **Hands cross**: Fingers of both hands meet in the middle (forming the "+")

---

## Finger Position (Each Hand)

```
        Index (8)  ↑  EXTENDED (up)
        Middle(12) ↑  EXTENDED (up)
        ─────────────────
        Ring (16)  ↓  FOLDED (into palm)
        Pinky (20) ↓  FOLDED (into palm)
        Thumb      →  relaxed
```

**Side view of one hand:**
```
     Index finger ────┐
     Middle finger ────┤  (both pointing UP/forward)
                       │
     Ring finger  ──────●  (curled into palm)
     Pinky       ──────●  (curled into palm)
```

---

## Hand Orientation

### Right Hand (your right)
- **Vertical**: wrist-to-index finger points mostly UP/DOWN
- Like holding your hand up with fingers pointing to the ceiling

```
    Right hand (vertical):
    
         ↑
         │  Index
         │  Middle
         │
    ─────●─────  wrist
```

### Left Hand (your left)
- **Horizontal**: wrist-to-index finger points mostly LEFT/RIGHT
- Like holding your hand sideways with fingers pointing to the side

```
    Left hand (horizontal):
    
    Index Middle
         │
    ─────●─────  wrist
```

---

## The Cross Formation (Top View)

Both hands form a **+** when viewed from the front:

```
         ↑
         │  Right hand fingers (vertical)
         │
    ─────┼─────
         │
    Left hand fingers (horizontal)
         │
```

**Or think of it as:**
```
    Left hand:  ────●────  (index+middle pointing right)
                        │
                        │  Right hand: index+middle pointing up
                        │
                        ●
```

**The fingers must be CLOSE** — left and right index/middle tips should be within ~2–3 cm of each other (forming the cross intersection).

---

## Step-by-Step

1. **Left hand**: Fold ring and pinky fingers into your palm. Extend index and middle fingers. Hold hand **horizontally** (fingers pointing to your right).
2. **Right hand**: Same finger position. Hold hand **vertically** (fingers pointing up).
3. **Bring them together**: Place the right hand’s fingers in front of/behind the left hand’s fingers so they cross.
4. **Hold in front of camera**: Both hands visible, facing the camera.
5. **Hold for ~0.2 seconds** — the effect needs a few frames to trigger.

---

## Quick Checklist

- [ ] Both hands visible
- [ ] Index + middle extended on BOTH hands
- [ ] Ring + pinky folded on BOTH hands
- [ ] Hands form a cross (+) — fingers meet
- [ ] Well-lit area
- [ ] Hold steady for a moment

---

## Troubleshooting

| Problem | Fix |
|--------|-----|
| Nothing happens | Ensure both hands are visible and clearly in frame |
| Triggers randomly | Reduce movement; hold the pose steady |
| Can't detect | Improve lighting; avoid busy backgrounds |
| Hand orientation wrong | Right hand: fingers up. Left hand: fingers sideways |
