# Panduan Hand Sign Kage Bunshin

Cara memosisikan tangan agar efek klon terpicu.

---

## Gambar Bentuk Tangan

### Satu Tangan (Tampak Depan — Telapak Menghadap Kamera)

```
                    ╭───╮
                    │   │  ← Telunjuk (TERULUR ke atas)
                    ╰───╯
                    ╭───╮
                    │   │  ← Jari tengah (TERULUR ke atas)
                    ╰───╯
    ╭───╮       ╭───┴───╮
    │   │       │       │  ← Jari manis & kelingking (DILIPAT)
    ╰───╯       ╰───────╯
     ↑
   Ibu jari (rileks ke samping)
```

### Satu Tangan (Tampak Samping)

```
    Telunjuk ─────┐
    Tengah   ─────┤  ↑ Keduanya lurus ke depan
                  │
    Manis    ══●══╯  ← Terlipat ke telapak
    Kelingking ●
```

### Bentuk Silang — Kedua Tangan Bersatu (Tampak Depan)

```
                    │
                    │  Tangan KANAN
                    │  (jari ke ATAS)
                    │
              ╭─────┼─────╮
              │     │     │
              │  ───●───  │  ← Titik persilangan
              │     │     │     (jari bertemu di sini)
              ╰─────┼─────╯
                    │
    ════════════════╪════════════════
                    │
         Tangan KIRI (jari ke SAMPING)
```

### Ilustrasi Posisi (Tampak dari Atas)

```
                    KAMERA
                       │
                       ▼
              ┌───────────────┐
              │               │
              │    ┌───┐      │
              │    │ ↑ │      │  Tangan kanan: jari ke atas
              │    │ ↑ │      │
              │ ───┼─●─┼───   │  ● = tempat jari bersilang
              │    │   │      │
              │    └───┘      │  Tangan kiri: jari ke samping (→)
              │               │
              └───────────────┘
```

---

## Ringkasan

Kamu perlu **KEDUA tangan** dalam frame, membentuk **bentuk silang (+)**:

- **Tangan kiri**: Jari telunjuk + jari tengah KE ATAS, jari manis + kelingking DILIPAT
- **Tangan kanan**: Posisi jari yang sama
- **Tangan bersilang**: Jari kedua tangan bertemu di tengah (membentuk "+")

---

## Posisi Jari (Setiap Tangan)

```
        Telunjuk (8)  ↑  TERULUR (ke atas)
        Tengah (12)   ↑  TERULUR (ke atas)
        ─────────────────
        Manis (16)    ↓  DILIPAT (ke telapak)
        Kelingking(20) ↓  DILIPAT (ke telapak)
        Ibu jari      →  rileks
```

**Tampilan samping satu tangan:**
```
     Jari telunjuk  ────┐
     Jari tengah   ────┤  (keduanya mengarah KE ATAS/depan)
                      │
     Jari manis   ──────●  (terlipat ke telapak)
     Kelingking   ──────●  (terlipat ke telapak)
```

---

## Orientasi Tangan

### Tangan Kanan
- **Vertikal**: pergelangan-ke-telunjuk mengarah KE ATAS/BAWAH
- Seperti mengangkat tangan dengan jari mengarah ke langit-langit

```
    Tangan kanan (vertikal):
    
         ↑
         │  Telunjuk
         │  Tengah
         │
    ─────●─────  pergelangan
```

### Tangan Kiri
- **Horizontal**: pergelangan-ke-telunjuk mengarah KE KIRI/KANAN
- Seperti memegang tangan menyamping dengan jari mengarah ke samping

```
    Tangan kiri (horizontal):
    
    Telunjuk Tengah
         │
    ─────●─────  pergelangan
```

---

## Bentuk Silang (Tampak Depan)

Kedua tangan membentuk **+** saat dilihat dari depan:

```
         ↑
         │  Jari tangan kanan (vertikal)
         │
    ─────┼─────
         │
    Jari tangan kiri (horizontal)
         │
```

**Atau bayangkan:**
```
    Tangan kiri:  ────●────  (telunjuk+tengah mengarah kanan)
                         │
                         │  Tangan kanan: telunjuk+tengah mengarah atas
                         │
                         ●
```

**Jari harus BERDEKATAN** — ujung telunjuk dan tengah kiri-kanan harus dalam jarak ~2–3 cm (membentuk persilangan).

---

## Langkah demi Langkah

1. **Tangan kiri**: Lipat jari manis dan kelingking ke telapak. Ulurkan jari telunjuk dan tengah. Pegang tangan **horizontal** (jari mengarah ke kananmu).
2. **Tangan kanan**: Posisi jari sama. Pegang tangan **vertikal** (jari mengarah ke atas).
3. **Satukan**: Letakkan jari tangan kanan di depan/belakang jari tangan kiri hingga bersilang.
4. **Tahan di depan kamera**: Kedua tangan terlihat, menghadap kamera.
5. **Tahan ~0,2 detik** — efek butuh beberapa frame untuk terpicu.

---

## Checklist Singkat

- [ ] Kedua tangan terlihat
- [ ] Telunjuk + tengah terulur di KEDUA tangan
- [ ] Manis + kelingking dilipat di KEDUA tangan
- [ ] Tangan membentuk silang (+) — jari bertemu
- [ ] Pencahayaan cukup
- [ ] Tahan posisi sebentar

---

## Pemecahan Masalah

| Masalah | Solusi |
|---------|--------|
| Tidak ada reaksi | Pastikan kedua tangan terlihat jelas dalam frame |
| Terpicu acak | Kurangi gerakan; tahan pose dengan stabil |
| Tidak terdeteksi | Perbaiki pencahayaan; hindari background ramai |
| Orientasi tangan salah | Tangan kanan: jari ke atas. Tangan kiri: jari ke samping |
